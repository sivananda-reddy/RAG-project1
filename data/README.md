Place your PDF files here for processing.

The RAG system will:
1. Load all .pdf files from this directory
2. Extract text content
3. Create embeddings
4. Store in vector database
5. Enable semantic search

You can place any of these:
- Research papers
- Technical documentation
- Books
- Articles
- Reports
- Any text-based PDFs

Note: Scanned image PDFs require OCR setup. Text-based PDFs work best.
